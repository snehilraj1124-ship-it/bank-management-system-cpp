# Bank Management System (C++)

## 📌 Project Overview

Bank Management System is a beginner-friendly console application developed using **C++ Programming**. The project demonstrates the fundamentals of Object-Oriented Programming (OOP) by allowing users to perform basic banking operations through a menu-driven interface.

The application enables users to create a bank account, deposit money, withdraw money, and display account details. The project is designed to strengthen programming concepts such as classes, objects, encapsulation, functions, conditional statements, loops, and user input handling.
